# API 명세서

## 개요

이 문서는 프로젝트의 API 명세를 정의합니다.

---

## 엔드포인트

| API 이름          | URL                          | Method | 설명                           |
|-------------------|------------------------------|--------|--------------------------------|
| **게시판 목록**   | `/posts`                     | GET    | 게시판 목록을 가져옵니다.      |
| **게시글 상세**   | `/posts/detail/<id>`         | GET    | 특정 게시글의 상세 정보를 가져옵니다. |
| **회원가입**      | `/user/signup`               | GET    | 회원가입 페이지를 렌더링합니다. |
| **로그인**        | `/user/login`                | GET    | 로그인 페이지를 렌더링합니다.   |
| **회원정보 수정** | `/user/edit-profile`         | GET    | 회원정보 수정 페이지를 렌더링합니다. |
| **요청 목록**     | `/admin-request`             | GET    | 관리자 요청 목록을 가져옵니다.  |
| **요청 상세**     | `/admin-request/detail/<id>` | GET    | 특정 관리자 요청의 상세 정보를 가져옵니다. |
